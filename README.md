# 📊 Excel Sales Report Automator

مشروع أتمتة تقارير المبيعات باستخدام لغة بايثون. يقوم النظام بمعالجة البيانات الخام وتحويلها إلى تقارير Excel جاهزة للإدارة تتضمن تحليلات بصرية وحسابية دقيقة.

## ✨ المميزات (Features)
تتبع المشروع نهجاً تدريجياً في الأتمتة يشمل:
1. **معالجة البيانات:** تحويل آلاف السجلات إلى جداول محورية (Pivot Tables) باستخدام `Pandas`.
2. **التحليل البصري:** إنشاء رسوم بيانية (Bar Charts) تلقائية باستخدام `Openpyxl`.
3. **العمليات الحسابية:** حقن معادلات Excel الحسابية (`SUM`) برمجياً لضمان تفاعلية التقرير.
4. **التنسيق الاحترافي:** تخصيص الخطوط، العناوين، وتنسيقات العملة لجعل التقرير جاهزاً للعرض.

## 🛠️ المكتبات المستخدمة (Tech Stack)
* **Python 3.x**
* **Pandas** (Data Manipulation)
* **Openpyxl** (Excel Engine)


## 📁 هيكل المشروع (Project Structure)
ينقسم الكود إلى مراحل لسهولة الفهم والتطوير:
* `1.make-pivot-table.py`: إنشاء الجداول المحورية.
* `2.add-charts.py`: إضافة الرسوم البيانية.
* `3.apply-formulas.py`: كتابة المعادلات الحسابية.
* `4.format-cells.py`: تنسيق المظهر النهائي.
* `5.pivot-to-report.py`: السكربت الشامل (Master Script).
